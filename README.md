## 👨‍💻 About Me

I am **Lochithya Hettiarachchi**, a Software Engineering undergraduate at the **University of Kelaniya**, with a strong interest in full-stack development, system design, database systems, UI/UX design, and practical software solutions.

I enjoy building meaningful applications that solve real-world problems, especially in areas such as healthcare, e-commerce, productivity, and digital transformation. My work focuses on creating clean user interfaces, structured backend systems, and database-driven applications.
